A command-line writing tool that strips away everything except the text. No formatting toolbar, no distraction mode (because there's nothing to distract from), no sync conflicts.

**Features:**
- Plain markdown storage, no database
- Live word count and writing session tracking
- Auto-save on every keystroke
- Git-backed version history

Built with Go. Inspired by the philosophy that a writer's tool should disappear when you're in flow.
